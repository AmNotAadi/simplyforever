System Requirements

Minimum Requirements

  • Minecraft Java Edition 1.20+
  • 2GB RAM allocated to Minecraft
  • Stable internet connection
  • Java 17 or higher

Recommended Requirements

  • Minecraft Java Edition 1.20+
  • 4GB+ RAM allocated to Minecraft
  • High-speed internet connection
  • Java 17 or higher
  • OptiFine or Sodium mod (optional)

Having Trouble?

Can't Connect?

Make sure you're using the correct IP address and have a stable internet connection. Try restarting Minecraft if the issue persists.

Version Mismatch?

Ensure you're using Minecraft Java Edition 1.20 or higher. Bedrock Edition is not supported.

Lag Issues?

Try allocating more RAM to Minecraft in your launcher settings. Close other applications to free up system resources.
